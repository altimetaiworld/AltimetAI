Privacy Policy

Last updated: June 28, 2026

1. Data Boundaries and VPC Isolation

Altimet AI designs and deploys custom systems. All production systems are built to reside inside the client's Virtual Private Cloud (VPC) boundary. No query records, document files, schema metadata, or model logs are collected or transmitted to external servers.

2. Zero Data Retention Policies

When routing queries to model providers (such as OpenAI, Anthropic, or custom private servers), we configure secure enterprise API nodes with strict Zero Data Retention (ZDR) options. Under no circumstances is your data utilized for model fine-tuning or secondary training runs.

3. Security Controls (SOC 2 Type II Alignment)

We enforce administrative, logical, and physical security measures designed to align with SOC 2 guidelines. Database schema connections are handled exclusively via secure read-only profiles with encryption at rest and in transit.
